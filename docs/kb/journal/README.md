# kb journals — header convention

One file per agent: `journal/<agent>.md`. First-person, append newest-at-top, don't rewrite
history. Read your OWN journal on session start; never edit another agent's. Durable tool
facts belong in the sheets (graded per `EVIDENCE.md`); project gotchas in `docs/LEARNINGS.md`
— the journal is for what _you_ did, tried, and suspect.

Header convention for a new journal:

    # Journal — <Agent name> (<which tool / sessions>)
    <one-line note on what this journal is>
    ---
    ## YYYY-MM-DD — <short title>
    <entry>
